ULISE Filter

Estimate zero-step state and one-step input delay

Description:

Use the ULISE Fliter block to estimate zero-step states and one-step input delay given kown input and measurement output vector. The implemnetation assumes a the plant is discrete-time, linear time-invariant (LTI) system generated by LTI Plant block.
