ULISE Filter

Estimate the zero-step state and one-step-delayed unknown input of a discrete-time linear time-invariant system

Description:

Use the ULISE Filter block to estimate zero-step state and unknown inputs with one-step delay using the measured outputs and known control inputs. In addition to the state and unknown input estimates, the block outputs their corresponding covariance matrices: the state error covariance , the unknown input covariance , and the cross-covariance .
The implementation assumes the plant is a discrete-time linear time-invariant (LTI) system generated by the LTI Plant.
The initial state estimate and the initial state error covariance are assumed to be given.
