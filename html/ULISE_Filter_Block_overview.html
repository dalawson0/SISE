ULISE Filter

Estimate zero-step state and one-step input delay

Description:

Use the ULISE Fliter block to estimate zero-step states and one-step input delay given kown input and measurement output vector. The implemnetation assumes a the plant is discrete-time, time-invariant linear system generated by LTI Plant block.

Limitations:

The system must satisfy the same conditions as the ULISE function for ULISE Filter to produce unbiased stable estimates.
(i) Strong detectability
(ii) Zero-step state, One-step Input delay
(iii) Frequency-Domain Identifiability ? What is this?
Formally, these conditions correspond to rank constraints on the system and noise matrices.
If these conditions are not met, ULISE Filter may fail to converge or may require estimating the unknown inputs with more than one step delay. In such cases, an error message is returned.
For the complete mathematical definitions, see the ULISE function documentation or the paper [1].

Examples:

  • ULISE_model_fault_identification
