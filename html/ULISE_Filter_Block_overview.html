ULISE Filter

Estimate zero-step state and one-step input delay

Description

Use the ULISE Fliter block to estimate zero-step states and one-step input delay given kown input and measurement output vector. The implemnetation assumes a the plant is discrete-time, time-invariant linear system.

If these conditions are not met, ULISE Filter may fail to converge or may require estimating the unknown inputs with more than one step delay. In such cases, an error message is returned.
