System Setup

Open the fault identification model initialization file for editing

Description:

All system parameters and simulation settings for the LTI Plant and ULISE Filter blocks are defined in the initialization script initFcn_ULISE_faultIdentification.m. This script is automatically executed when the ULISE Fault Identification example model is run, ensuring consistent/updated parameterization.
In general, a discrete-time LTI system with n states , m known inputs , p unknown inputs , and l measured outputs , the file must define the following structures and fields:

Simulation Setup struct (SimSetup):

Name
Description
Ts
Simulation sample time (default 1 second for discrete-time systems)
K
Total time steps

Known and Unknown Input
The LTI Plant block supports three methods for providing the known input and unknown input .
  • Batch Input via MATLAB Matrices:
For batch workflows, InitFcn_ULISE.m defines the input matrices u and d, where each column corresponds to the inputs at a given time step. These matrices can be used with either Signal Editor or From Workspace blocks.
Note: if there are no known inputs, i.e., , the initialization file replaces the input (and the corresponding B and D matrices) with a scalar dummy zero input for ease of implementation.
  • External Hardware Inputs:
To supply signals from external hardware, comment out the default u and d definitions through the end of the file.

Actions:

Opens the InitFcn_ULISE.m script.
