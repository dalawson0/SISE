The LTI Plant block supports several methods for supplying the known input and unknown input . In batch-defined workflows, these signals are provided as matrices and , where each column corresponds to the input or at time k. These matrices can be supplied to the model using either a Signal Editor blockallowing users to view and edit input scenarios interactivelyor a From Workspace block, which reads timeseries objects directly from MATLAB. Alternatively, the LTI Plant can accept input signals from external hardware sources for real-time operation.
