SISE Toolbox

Simultaneous Input and State Estimation for Linear Systems

The SISE Toolbox provides a MATLAB implementation of algorithms for estimating both the state and unknown inputs of linear systems. It is designed to support research, simulation, and practical applications in control and signal processing.
