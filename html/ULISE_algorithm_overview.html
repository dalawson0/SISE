Example

ULISE for Fault Identification

Inputs

A state transition matrix
B control-input matrix
C measurement matrix
D measurement feedthrough from u
G disturbance-to-state matrix
H disturbance feedthrough to output
Q process noise covariance
R measurement noise covariance
K number of time steps (positive)
u known control inputs (each column = u_k)
y measured outputs (each column = y_k)
xhat0 initial state estimate (xhat_{0|0})
P_x0 initial state error covariance (Px_{0|0})

Outputs

xhat_UL state estimates at each k (xhat_{k|k})
Px_UL state error covariance at each k (Px_{k|k})
dhat_UL unknown input estimates (d2 delayed by one time step)
Pd_UL unknown input error covariance at each k
Pxd_UL cross-covariance between x and d

Assumptions

  • System is linear, time-invariant, and discrete-time.
  • Input arrays u and y must have K columns. If u or y are single column vectors, set K = size(u, 2).
  • Initial state estimate and covariance are provided.
  • Process and measurement noises are zero-mean, white Gaussian noise.
  • The process and measurement noise covariance must be positive (semi-)definite matrices i.e. and .
