Getting Started

Exploring the Examples

To quickly understand the capabilities of the SISE toolbox, we recommend exploring the provided examples. A good starting point is the Fault Identification example, which demonstrates how to implement the algorithm on simulated data step-by-step.

Simulate Your Own System

To create a custom simulation:
  1. Define a specific plant that meets your desired criteria.
  2. Use the Save As feature to creat your own copy of an existing example.
  3. Modify the relevant parameters and save your updated version.
This approach allows you to experiment with different configurations while leveraging the existing examples as templates.
