Getting Started

Exploring Examples

To quickly understand the capabilities of the SISE toolbox, we recommend exploring the provided examples. A good starting point is the Fault Identification script or the Fault Identification model example, which demonstrates how to implement the ULISE algorithm on simulated data.

Simulate Your Own System

Custom simulations can be created by modifying existing examples:
  1. Define a specific plant that meets your desired criteria.
  2. Use the Save As feature to create your own copy of an existing example.
  3. Modify the relevant parameters and save your updated version.
This workflow allows you to experiment with different configurations while leveraging the existing examples as templates.
